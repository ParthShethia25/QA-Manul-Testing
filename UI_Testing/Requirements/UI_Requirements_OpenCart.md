1. User Management
   - Users should be able to register using email, password and basic profile details.
   - Login should authenticate users with valid credentials.
   - Forgot password flow should send a reset link (simulated).

2. Product Catalog
   - Products should display name, price, stock status and image.
   - Users should be able to filter and sort products by category, price and rating.
   - Product detail page should show description, reviews and add-to-cart option.

3. Cart & Checkout
   - Users should be able to add/remove items from cart.
   - Quantity changes should update line totals and cart totals.
   - Checkout flow should collect shipping address, select payment and show order confirmation.

4. Orders & Account
   - Users should be able to view order history.
   - Users can update profile and addresses.
   - Logout should end the session and redirect to homepage.

5. UI/Usability
   - All pages must be responsive on desktop view.
   - Error messages should be clear and consistent.
   - Mandatory fields should show validation messages when empty or invalid.
